API Documentation

Complete documentation voor de League of Legends Champions en Items API.

Base URL: http://localhost:3000

Champions Endpoints

GET /api/champions

Haal alle champions op met pagination, search en filtering.

Query Parameters:

Parameter Type Beschrijving
limit integer Aantal resultaten (default: 10)
offset integer Aantal overslaan (default: 0)
search string Zoek op naam of titel
role string Filter op role
sort string Sorteer op veld (name, difficulty, release_date)
order string asc of desc

GET /api/champions/:id

Haal één champion op.

POST /api/champions

Voeg nieuwe champion toe.

Required Fields:

  • name - String (alleen letters)
  • title - String
  • role - Tank, Fighter, Assassin, Mage, Marksman, Support
  • difficulty - Integer (1-10)
  • release_date - String (YYYY-MM-DD, moet in verleden)
  • lore - String (optioneel)
  • image_url - String (optioneel)

PUT /api/champions/:id

Update bestaande champion. Zelfde velden als POST.

DELETE /api/champions/:id

Verwijder champion.

Items Endpoints

GET /api/items

Haal alle items op. Zelfde query parameters als champions.

GET /api/items/:id

Haal één item op.

POST /api/items

Voeg nieuw item toe.

Required Fields:

  • name - String
  • description - String
  • cost - Integer (>= 0)
  • category - Damage, Defense, Magic, Movement, Consumable
  • stats - String (optioneel)
  • image_url - String (optioneel)

PUT /api/items/:id

Update item.

DELETE /api/items/:id

Verwijder item.
