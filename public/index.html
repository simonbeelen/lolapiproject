League of Legends API

Welkom bij de League of Legends Champions en Items API. Deze API biedt endpoints om champions en items te beheren.

Base URL: http://localhost:3000

Champions Endpoints

GET /api/champions

Haal alle champions op. Ondersteunt pagination, zoeken en filtering.

Query Parameters:

Parameter Type Beschrijving Voorbeeld
limit integer Aantal resultaten per pagina (default: 10) ?limit=5
offset integer Aantal resultaten om over te slaan (default: 0) ?offset=10
search string Zoek op naam of titel ?search=Yasuo
role string Filter op role ?role=Assassin
sort string Sorteer op veld (default: name) ?sort=difficulty
order string Sorteer richting: asc of desc (default: asc) ?order=desc

Voorbeeld Response:

{
  "data": [
    {
      "id": 1,
      "name": "Yasuo",
      "title": "the Unforgiven",
      "role": "Assassin",
      "difficulty": 8,
      "release_date": "2013-12-13",
      "lore": "An Ionian of deep resolve...",
      "created_at": "2024-12-22 10:00:00"
    }
  ],
  "pagination": {
    "total": 50,
    "limit": 10,
    "offset": 0,
    "total_pages": 5
  }
}

GET /api/champions/:id

Haal één specifieke champion op aan de hand van ID.

Voorbeeld:

GET /api/champions/1

Response:

{
  "id": 1,
  "name": "Yasuo",
  "title": "the Unforgiven",
  "role": "Assassin",
  "difficulty": 8,
  "release_date": "2013-12-13",
  "lore": "An Ionian of deep resolve...",
  "created_at": "2024-12-22 10:00:00"
}

POST /api/champions

Voeg een nieuwe champion toe aan de database.

Request Body:

Veld Type Verplicht Validatie
name string Ja Alleen letters, niet leeg
title string Ja Niet leeg
role string Ja Tank, Fighter, Assassin, Mage, Marksman, Support
difficulty integer Ja 1-10
release_date string Ja YYYY-MM-DD formaat, moet in verleden liggen
lore string Nee -

Voorbeeld Request:

{
  "name": "Zed",
  "title": "the Master of Shadows",
  "role": "Assassin",
  "difficulty": 7,
  "release_date": "2012-11-13",
  "lore": "The first ninja in 200 years..."
}

PUT /api/champions/:id

Update een bestaande champion.

Request Body:

Zelfde structuur als POST endpoint.

Voorbeeld:

PUT /api/champions/1

DELETE /api/champions/:id

Verwijder een champion uit de database.

Voorbeeld:

DELETE /api/champions/1

Items Endpoints

GET /api/items

Haal alle items op. Ondersteunt pagination, zoeken en filtering.

Query Parameters:

Parameter Type Beschrijving Voorbeeld
limit integer Aantal resultaten per pagina (default: 10) ?limit=5
offset integer Aantal resultaten om over te slaan (default: 0) ?offset=10
search string Zoek op naam of beschrijving ?search=Blade
category string Filter op category ?category=Damage
sort string Sorteer op veld (default: name) ?sort=cost
order string Sorteer richting: asc of desc ?order=desc

Voorbeeld Response:

{
  "data": [
    {
      "id": 1,
      "name": "Infinity Edge",
      "description": "Massively increases critical strike damage",
      "cost": 3400,
      "category": "Damage",
      "stats": "+70 Attack Damage, +20% Critical Strike Chance",
      "created_at": "2024-12-22 10:00:00"
    }
  ],
  "pagination": {
    "total": 30,
    "limit": 10,
    "offset": 0,
    "total_pages": 3
  }
}

GET /api/items/:id

Haal één specifiek item op aan de hand van ID.

Voorbeeld:

GET /api/items/1

POST /api/items

Voeg een nieuw item toe aan de database.

Request Body:

Veld Type Verplicht Validatie
name string Ja Niet leeg
description string Ja Niet leeg
cost integer Ja Moet een getal zijn, >= 0
category string Ja Damage, Defense, Magic, Movement, Consumable
stats string Nee -

Voorbeeld Request:

{
  "name": "Rabadon's Deathcap",
  "description": "Increases ability power",
  "cost": 3600,
  "category": "Magic",
  "stats": "+120 Ability Power, +35% Ability Power"
}

PUT /api/items/:id

Update een bestaand item.

Request Body:

Zelfde structuur als POST endpoint.

DELETE /api/items/:id

Verwijder een item uit de database.

Voorbeeld:

DELETE /api/items/1

Error Responses

400 Bad Request - Validatie Error:

{
  "error": "Validation failed",
  "details": [
    "Name is required",
    "Difficulty must be between 1 and 10"
  ]
}

404 Not Found:

{
  "error": "Champion not found"
}

500 Internal Server Error:

{
  "error": "Something went wrong!"
}
